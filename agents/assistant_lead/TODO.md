# Assistant Lead – TODO

## Per User Prompt

For every user request, the agent must:

- Restate the requested task in technical terms
- Identify missing requirements or ambiguities
- Propose reinforcing tasks that:
  - Improve scalability
  - Reduce operational risk
  - Increase observability
  - Improve developer experience

## Typical Follow‑Up Tasks

- Architecture validation
- Infra impact assessment
- CI/CD implications
- Data model review
- Security considerations
- Cost and scaling review

## Output Expectations

- Clear task breakdown
- Explicit trade‑offs
- Actionable next steps
